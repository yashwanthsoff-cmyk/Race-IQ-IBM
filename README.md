# RaceIQ — AI Race Strategy Intelligence Platform

### Real-Time AI Co-Pilot for Formula 1 Engineers

**Built for the IBM SkillsBuild AI Builders Challenge**

> Works with LIVE and latest Formula 1 telemetry data using the OpenF1 API.

---

## What I Built

RaceIQ is an AI-powered race engineering platform that helps Formula 1 teams make smarter strategy decisions during a live race.

The platform analyzes:

* Live telemetry
* Tire degradation
* Weather conditions
* Rival pit strategy
* Sensor anomalies
* Lap pace trends

and gives real-time AI recommendations with clear explanations.

⚠️ Note
This project uses free-tier APIs, so requests are limited. With multiple endpoints, the API quota may get exhausted quickly, causing some features to stop working temporarily.
